פישוט מעריכים רציונליים

הבסיס b מורם בחזקת n/m שווה ל:

bn/m = (m√b)n = m√(bn)

דוגמא:

הבסיס 2 שהועלה בחזקת 3/2 שווה ל-1 חלקי הבסיס 2 המועלה בחזקת 3:

23/2 = 2√(23) = 2.828

פישוט שברים עם אקספוננטים

שברים עם אקספוננטים:

(a / b)n = an / bn

דוגמא:

(4/3)3 = 43 / 33 = 64 / 27 = 2.37

פישוט מעריכים שליליים

הבסיס b המורם בחזקת מינוס n שווה ל-1 חלקי הבסיס b המועלה בחזקת n:

b-n = 1 / bn

דוגמא:

הבסיס 2 שהועלה בחזקת מינוס 3 שווה ל-1 חלקי הבסיס 2 המועלה בחזקת 3:

2-3 = 1/23 = 1/(2⋅2⋅2) = 1/8 = 0.125

פישוט רדיקלים עם מעריכים

עבור רדיקלי עם אקספוננט:

(m√a)n = an/m

דוגמא:

(√5)4 = 54/2 = 52 = 25
